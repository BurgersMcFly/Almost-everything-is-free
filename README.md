# Almost everything is free

Can get almost everything for free (weapons, cyberware, attachments, materials, ammo, grenades, inhalers etc).

## Mod page on Nexus Mod

https://www.nexusmods.com/cyberpunk2077/mods/6241

## Installation

1. Download [redscript](https://www.nexusmods.com/cyberpunk2077/mods/1511).
2. Download the mod and drop its content in your game's folder. 
3. Should look like this: \Cyberpunk 2077\r6\scripts\AlmostEverythingIsFree.reds

## Notes

You might need [cybercmd](https://www.nexusmods.com/cyberpunk2077/mods/5176) for compatibility with redmod.
